\# Excel 文本批量情感分类工具



\## 项目简介

一个基于 Python 的轻量级文本情感分析工具。读取 Excel 文件中的文本数据，通过关键词匹配进行情感倾向判断（正面/负面/中性），并将结果自动写回 Excel。



\## 使用场景

适用于需要对大量短文本进行快速情感分类的场景，如用户评论、产品反馈、问卷调查等。手动处理耗时且主观，本工具可将处理时间从数十分钟缩短至数秒。



\## 技术栈

\- Python 3

\- Pandas

\- OpenPyXL
